HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • "H" elements are used to designate headers.
  • HTML serves as the structural base for websites.
  • Using "section" will not display content unless other elements are added.
  • "Link" elements can be used to apply external code (such as a CSS file) to your HTML code.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • CSS is used to stylize your website.
  • CSS code can be implemented by inline, internal, or external methods.
  • A CSS rule is composed of a selector, and a declaration block.
  • A declaration is composed of a property and a value.

Git

  • "git status": checks what branch we are currently on.
  • "git checkout -b branch-name": creates a new branch and switches to it.
  • "CD .." is used to navigate upwards in a file directory.

JavaScript

  • Javascript is used to add interactivity and functionality to webpages.
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • An array is a named container that stores a number of variables.
  • A function must be called for the code within it to process.
